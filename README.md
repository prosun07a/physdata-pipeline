# PhysData-Pipeline: Research Automation & Visualization
**Developed by Prosun Kanti Datta (HSC 2027)** ## 🔬 Project Overview
PhysData-Pipeline is a personal project developed to automate the processing and visualization of experimental physics data. The goal of this suite is to handle the transition from raw laboratory datasets (Excel/CSV) to high-fidelity, publication-standard figures, specifically optimized for research workflows in Materials Physics and Nanotechnology.

## 🚀 Key Features
- **Data Ingestion & Cleaning:** Python-based scripts to process multi-column experimental datasets.
- **Scientific Graphing:** Generates high-resolution (300 DPI) plots using Matplotlib with LaTeX-formatted axis labels and mathematical notations.
- **Statistical Analysis:** Built-in modules for linear regression, curve fitting, and data normalization.
- **Project Dashboard:** A professional web-based interface to showcase automated data outputs.

## 🛠️ Technical Stack
- **Languages:** Python (Pandas, NumPy, Matplotlib), C++
- **Web Frontend:** HTML5, Tailwind CSS, GSAP (for interactive data presentation)
- **Tools:** MS Excel (Advanced Data Processing), LaTeX

## 📂 Repository Structure
- `/scripts`: Core Python logic for automated analysis.
- `/data`: Sample physics datasets for testing.
- `/output`: Exported scientific figures.
- `index.html`: Web-based data showcase.
